# Go Task
## Resumo
+ Kanbam de tarefas com Angular, Tailwind e persistência local.
+ Figma: [Link](https://www.figma.com/design/rX2D4XX2V5Wq3QtTQOCUh1/GoTask--Community-?node-id=0-1&p=f&t=1jTH1GsqbMbxrN4H-0)

## Conceitos utilizados
1. Componentização.
2. Separação de responsabilidades entre services e componentes.
3. Desacoplamento entre componentes.
4. Padrão de projeto.
5. Fluxo de dados entre componentes.
6. Gerenciamento de estado(fonte de verdade).
7. Imutabilidade de dados.
8. Padronização de nomes e organização de arquivos.
9. Mobile First e Tailwind.
10. Angular Material CDK(Dialog e Drag and Drop).
11. RXJS(Behaviour Subject).
12. Pipes (AsyncPipe, SlicePipe).
13. @let, @if, @for.
14. Decorators(@Input).
15. Injeção de dependências (Services).
16. Interpolação, Property Binding e Event Binding.

## Flxuo do projeto
1. Criação do projeto inicial.
2. Criação do layout dos componentes.
3. Criação do fluxo de dados.
4. Criação da lógica dos componentes.
5. Persistindo as tarefas em LocalStorage.
   
## Observação importante
O projeto usa angular 19. Num próximo projeto/módulo de estudo será visto o novo angular 22.
