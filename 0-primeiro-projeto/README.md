# Projeto 0
## Resumo
Há apenas uma exploração das pastas do Angular e do uso do Angular CLI. Não construção de qualquer site. É **SOMENTE** um primeiro projeto em que pude entender como a CLI é utilizada.

## Comandos base
### Criar componente
```bash
npm run ng generate c nome-do-componente
```
Ou
```bash
npm run ng generate component nome-do-componente
```
### Criar componente sem testes
```bash
npm run ng generate c nome-do-componente --skip-tests=true
```
### Criar componente na pasta específica dentro de src
```bash
npm run ng generate c components/nome-do-componente
```
### Criar service
```bash
npm run ng generate service services/nome-do-service
```
